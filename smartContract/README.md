# Hogwarts Smart Contract

ERC-1155 Smart Contract that provides minting functionality of 4 different types of tokens:

 1. GRYFFINDOR 
 2. HUFFLEPUFF
 3. SLYTHERIN
 4. RAVENCLAW 

## Installation

Make sure you have NPM installed.

Install hardhat

```bash
 $ npm install --save-dev hardhat
```
Compile smart contracts

 ```bash
 $ npx hardhat compile
```
Test Smart Contract
 ```bash
 $ npx hardhat test
```
